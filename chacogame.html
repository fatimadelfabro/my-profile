ChacoGame

The application was built along to other two developers.

This was made as a final project of the course Programación y Desarrollo Web.

It is a game in which you must to answer questions about Chaco's history.

The technologies used behind the scenes are :

  • Django Framework (working with Python)
  • SQLite like database.

Besides, we used the MVC(Model View Controller) pattern provided by Django.

If you have accepted the challenge. You can play here.
